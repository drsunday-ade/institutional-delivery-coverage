# institutional-delivery-coverage

Reproducible code and outputs for a global country-year fixed-effects analysis of institutional delivery coverage and maternal mortality.

## Manuscript

**Institutional delivery coverage and maternal mortality: a global country-year fixed-effects analysis**

**Authors:** Sunday A. Adetunji, MD, MPH; Oluwatimileyin C. Adetunji, BSc, PMP; Rhoda O. Oyewusi, RN, RM

**Manuscript status:** under review at *The Lancet Global Health*.

## Repository contents

- `data/` — raw WHO Global Health Observatory indicator files used in the analysis
- `scripts/analytics_mmr_facilitydel.R` — main reproducible analysis script
- `scripts/deploy_shinyapp.R` — dashboard deployment script
- `output/figures/` — manuscript figures
- `output/tables/` — manuscript tables
- `output/logs/` — reproducibility logs
- `output/shiny_app/` — dashboard assets and supporting files

## Data source

WHO Global Health Observatory (GHO), Maternal and reproductive health topic portal:  
https://www.who.int/data/gho/data/themes/topics/topic-details/GHO/maternal-and-reproductive-health

Archive generation date used for the analysis: **2025-10-06**

## Reproducibility

This repository contains the code and outputs used to generate the analytic dataset, primary models, robustness analyses, figures, tables, and policy translation outputs reported in the manuscript.
